#조건문
if와 elif의 차이점 :: if ~ elif 구조는 앞의 조건이 참이면 뒤의 조건문은 더 이상 실행하지 않음. 하지만 if 문으로만 쓰여있다면 참, 거짓 여부와 관계없이 항상 실행하게 되는 것
